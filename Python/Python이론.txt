Python

컴퓨터에게 자세한 명령어 instruction -> 임베디드 프로그램 embedded program 
컴파일러 compiler : 통역기


인터프리터 언어 ( 해석기) 
실행 전 컴파일할 필요가 없다.

한 줄의 명령어를 : 문장 statement 이라고 한다
문장들은 파이썬 인터프리터에 의하여 순차적으로 실행된다

문자열 String :  큰 다옴표나 작은 따옴표 안에 들어 있는 텍스트 데이터 
반드시 따옴표가 있어야 함.
print(Hello)
	SyntaxError : invalid syntax

명령어를 한 줄씩 입력하여 실행하는 것을 인터프리트 모드라고 함 
한 줄의 명령어를 입력하여 실행하고 결과를 즉시 알 수 있으며 현재 상태를 언제든 파악 가능 


명령어들이 저장된 파일을 소스 파일(source file)  (확장자 .py)


> 변수
명령어들은 위에서 아래로 순차적 실행
변수 = 변수에 저장할 값

>터틀 그래픽
카테시안 공간에서 커서 ( 터틀)을 이용하여 그림을 그리는 기능
